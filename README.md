# Inverse Physics-informed Neural Networks Procedure for Detecting Noise in Open Quantum Systems

This repository contains the code and supplementary materials for the article **"Inverse Physics-informed neural networks procedure for detecting noise in open quantum systems"**.

## Folder Structure

- **/data**  
  Contains datasets used for parameter find in the simulation of the PINN models.

- **Fuction file**  
  All fucion that you need for simulation and creat a data, including PINN models and auxiliary functions for training and evaluation.

- **Analise_dos_resultados files**  
  Images and figures used in the article, and result visualizations.

---

Feel free to explore the folders to understand the implementation and experiments carried out in the context of the article.
